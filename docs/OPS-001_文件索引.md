## 重要路徑

| 項目 | 路徑 |
|------|------|
| 後端程式 | `/var/www/mimeet/backend` |

| Admin dist | `/var/www/mimeet/admin/dist` |
| Nginx 設定 | `/etc/nginx/sites-available/mimeet` |
| Supervisor | `/etc/supervisor/conf.d/mimeet.conf` |
| Laravel Log | `/var/www/mimeet/backend/storage/logs/` |
| 更新腳本 | `/usr/local/bin/mimeet-update` |
