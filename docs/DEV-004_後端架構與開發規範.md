## 4. Controller 設計規範

### 4.1 Controller 標準結構

```php
<?php
namespace App\Http\Controllers\Api\V1\User;

use App\Http\Controllers\Controller;
use App\Http\Requests\User\UpdateProfileRequest;
use App\Services\User\UserService;
use App\Http\Resources\UserResource;
use Illuminate\Http\JsonResponse;

class ProfileController extends Controller
{
    public function __construct(
        private readonly UserService $userService
    ) {}

    /**
     * 取得當前用戶資料
     */
    public function me(): JsonResponse
    {
        $user = auth()->user()->load(['verificationRecords', 'subscription']);
        return response()->json(['data' => new UserResource($user)]);
    }

    /**
     * 更新個人資料
     */
    public function update(UpdateProfileRequest $request): JsonResponse
    {
        $user = $this->userService->updateProfile(
            auth()->user(),
            $request->validated()
        );
        return response()->json(['data' => new UserResource($user)]);
    }
}
```

**規範：**
- Controller 只負責：接收請求 → 呼叫 Service → 回傳 Response
- 業務邏輯一律放在 Service 層，Controller 不寫業務邏輯
- 使用 Form Request 做輸入驗證
- 使用 API Resource 做輸出格式化
- 依賴注入透過 Constructor 注入（readonly）

## 5. Service 設計規範

### 5.1 CreditScoreService 範例

```php
<?php
namespace App\Services\User;

use App\Models\User;
use App\Models\CreditScoreHistory;
use App\Events\CreditScoreChanged;
use Illuminate\Support\Facades\DB;

class CreditScoreService
{
    /**
     * 調整誠信分數
     *
     * @param  int    $delta    正數為加分，負數為扣分
     * @param  string $reason   變更原因（human-readable）
     * @param  string $type     觸發類型（date_verify/report/admin/...）
     */
    public function adjust(User $user, int $delta, string $reason, string $type): void
    {
        DB::transaction(function () use ($user, $delta, $reason, $type) {
            $oldScore = $user->credit_score;
            $newScore = max(0, min(100, $oldScore + $delta)); // 限制在 0-100

            $user->update(['credit_score' => $newScore]);

            CreditScoreHistory::create([
                'user_id'   => $user->id,
                'old_score' => $oldScore,
                'new_score' => $newScore,
                'delta'     => $delta,
                'reason'    => $reason,
                'type'      => $type,
            ]);

            event(new CreditScoreChanged($user, $oldScore, $newScore));
        });
    }
}
```

### 5.2 CreditScoreObserver（自動停權）

```php
<?php
namespace App\Observers;

use App\Models\User;
use Illuminate\Support\Facades\Mail;
use App\Mail\AccountSuspendedMail;

class CreditScoreObserver
{
    public function updated(User $user): void
    {
        // 分數 ≤ 0 且原本非停權狀態 → 自動停權
        if ($user->isDirty('credit_score')
            && $user->credit_score <= 0
            && $user->getOriginal('status') !== 'suspended')
        {
            $user->updateQuietly(['status' => 'suspended']);
            Mail::to($user->email)->queue(new AccountSuspendedMail($user));
        }

        // 分數 ≥ 30 且原本是停權（因分數歸零）→ 自動解停
        if ($user->isDirty('credit_score')
            && $user->credit_score >= 30
            && $user->status === 'suspended'
            && $user->getOriginal('credit_score') <= 0)
        {
            $user->updateQuietly(['status' => 'active']);
        }
    }
}
```

## 8. 關鍵業務邏輯實作說明

### 8.1 Ticket 案號生成

```php
// app/Services/Report/ReportService.php
// 格式：R{YYYYMMDD}{5位序號}  ex: R202401150001
public function generateTicketNumber(): string
{
    $date = now()->format('Ymd');
    $key = "ticket_seq_{$date}";

    // Redis 原子自增，確保每日序號唯一
    $seq = Redis::incr($key);
    if ($seq === 1) {
        Redis::expire($key, 86400 * 2); // 2天後過期
    }

    return 'R' . $date . str_pad($seq, 5, '0', STR_PAD_LEFT);
}
```

### 8.2 訪客記錄去重（24小時）

```php
// app/Services/User/UserVisitorService.php
public function recordVisit(User $visitedUser, User $visitor): void
{
    // VIP 隱身模式：不留記錄
    if ($visitor->privacy_settings['stealth_mode'] ?? false) {
        return;
    }

    // 不記錄自己瀏覽自己
    if ($visitedUser->id === $visitor->id) {
        return;
    }

    // UPSERT：同一對用戶只保留最新一筆，更新 visited_at
    UserProfileVisit::upsert(
        [
            'visited_user_id' => $visitedUser->id,
            'visitor_user_id' => $visitor->id,
            'visited_at'      => now(),
        ],
        uniqueBy: ['visited_user_id', 'visitor_user_id'],
        update:   ['visited_at']
    );
}
```

```php
// app/Services/Chat/AnonymousChatService.php
// 同一用戶在同一頻道始終顯示相同代號
public function getAnonAlias(int $userId, int $channelId): string
{
    // CRC32 保證：相同輸入 → 相同數字
    $num = abs(crc32("{$userId}_{$channelId}")) % 9999 + 1;
    return sprintf('神秘訪客#%04d', $num);
}
```

```php
protected function schedule(Schedule $schedule): void
```

```php
<?php

namespace App\Services;

use Illuminate\Http\Client\Response;
use Illuminate\Support\Facades\Http;
use Illuminate\Support\Facades\Log;

class ImageModerationService
{
    /**
     * 審核圖片是否符合規範
     *
     * @param  string  $imageUrl  已上傳至 DO Spaces 的公開 URL
     * @return bool  true = 通過，false = 違規
     */
    public function isImageSafe(string $imageUrl): bool
    {
        // 停用開關（開發 / 測試環境可關閉）
        if (! config('services.sightengine.enabled')) {
            return true;
        }

        try {
            $response = Http::timeout(10)->get(config('services.sightengine.endpoint'), [
                'url'        => $imageUrl,
                'models'     => 'nudity-2.1',  // 最新裸露偵測模型
                'api_user'   => config('services.sightengine.api_user'),
                'api_secret' => config('services.sightengine.api_secret'),
            ]);

            if (! $response->successful()) {
                // API 失敗時預設放行，避免阻擋正常上傳，並記錄告警
                Log::warning('Sightengine API error', [
                    'status' => $response->status(),
                    'body'   => $response->body(),
                    'url'    => $imageUrl,
                ]);
                return true;
            }

            $data = $response->json();

            // Sightengine nudity-2.1 回傳結構：
            // { "nudity": { "sexual_activity": 0.01, "sexual_display": 0.02, "erotica": 0.05, "suggestive": 0.1, "none": 0.92 } }
            $threshold = config('services.sightengine.threshold');
            $nudity    = $data['nudity'] ?? [];

            $maxScore = max(
                $nudity['sexual_activity'] ?? 0,
                $nudity['sexual_display']  ?? 0,
                $nudity['erotica']         ?? 0,
            );

            return $maxScore < $threshold;

        } catch (\Throwable $e) {
            // 網路異常時預設放行，記錄錯誤
            Log::error('Sightengine exception', [
                'message' => $e->getMessage(),
                'url'     => $imageUrl,
            ]);
            return true;
        }
    }
}
```

#### 13.5.1 app.mode 切換機制

#### 13.5.3 敏感設定儲存策略

| 設定類型 | 儲存位置 | 加密 |
|---------|---------|------|
| `app.mode` / `sms.provider` / `mail.host` 等非敏感 | `system_settings` 表 | 無 |
| `sms.*.password` / `mail.password` / `db.password` | `.env` 檔案 | AES-256 ✅ |

規則：GET 時密碼永遠回傳 `****`；PATCH 時密碼欄位為空則保留現有值。

#### 13.5.4 SystemSetting Model Helper

```php
class SystemSetting extends Model
{
    protected $table = 'system_settings';
    protected $primaryKey = 'key_name';
    public $incrementing = false;

    public static function get(string $key, mixed $default = null): mixed
    {
        return Cache::remember("sys:{$key}", 60, function () use ($key, $default) {
            return static::where('key_name', $key)->value('value') ?? $default;
        });
    }

    public static function set(string $key, mixed $value): void
    {
        static::updateOrCreate(['key_name' => $key], ['value' => (string) $value]);
        Cache::forget("sys:{$key}");
    }
}
```
